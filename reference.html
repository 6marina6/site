Рис. 1 - Архитектура серверного оборудования

Критерии построения ЛВС
  • Использование оборудования уровня Enterprise
  • Резервирование каналов связи
  • Сегментация сети на VLAN
  • Мониторинг трафика в реальном времени
Безопасность Wi-Fi
Рекомендуемые настройки:
  • Шифрование: WPA3-Enterprise
  • Скрытие SSID
  • MAC-фильтрация
  • Гостевой доступ через отдельную VLAN
Виртуализация сети

Используемые технологии:

  • VMware NSX
  • Контейнеризация Docker
  • Программно-определяемые сети (SDN)
Расположение оборудования
  • Серверная комната: 3 этаж, комната 305
  • Температурный режим: 18-22°C
  • ИБП: APC Smart-UPS 5000VA
  • Резервный генератор: Cummins 100кВт
